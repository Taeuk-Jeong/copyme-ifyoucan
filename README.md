# COPY ME IF YOU CAN

## 실행방법 (dev)

1. 각 packages디렉토리로 이동하여 script를 실행합니다

```bash
$ cd packages/client
$ yarn start:dev
```

2. root 디렉토리에서 설정된 script를 실행합니다 (package.json에 작성되어있음)

```bash
$ yarn client:start
$ yarn server:dev
```

## 개발환경 셋팅

```bash
# git clone
$ git clone https://github.com/krafton-jungle-project/copyme-ifyoucan.git

# install yarn
$ brew install yarn

# install nvm
$ brew install nvm
$ nvm install 16.19.0
$ nvm use 16.19.0
```
